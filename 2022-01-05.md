4. !! : 다른 타입의 데이터를 boolean 타입으로 명시적으로 형변환하기 위해 사용
    undefined, "", 0일 경우 결과 false
    그 외는 모두 true이다. (문자열 0은 true)
    ex)
        a    :: undefined
        !a   :: true
        !!a  :: false

        b    :: true
        !b   :: false
        !!b  :: true

        c    :: null
        !c   :: true
        !!c  :: false

        적용 전]
        var data = "1";
        if(data != undefined && data != "")
        {
        //실행 함수
        }

        적용 후]
        var data = "1";
        if(!!data)
        {
        //실행 함수
        }
